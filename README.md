ESP6288 / WM-386 sketch with Generic VFO functionality.

Sketch will compile on Arduino IDE and VSC.

Hardware consits of WM-386, SI5351 and SSD1306 .91"OLED. The VFO is accessable through a Webservice and has a build in Calibration routine.
The WiFi SSID and PASSWORD have to be specified in the Sketch.

The program has been created as PoC for using with a TR-2200GX but can be altered to any need you have.
